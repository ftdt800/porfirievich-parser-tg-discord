### Установка и запуск
1. Напишите СВОЙ хэш Telegram и идентификатор (проверьте на my.telegram.com )
2. Откройте telegram, dicord или txt-файл
3. Скопируйте случайный текст (только ctrl + c)
4. Подождите 2-5 секунд
5. Успех!
